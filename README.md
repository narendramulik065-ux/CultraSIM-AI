# 🌍 CulturaSim – AI-Based Cross-Cultural Conversation Simulator

## 📌 Overview

CulturaSim is an AI-powered platform designed to help students and professionals practice conversations in different cultural and professional environments.

Many talented individuals miss global opportunities not because of lack of knowledge, but because they struggle with communication in cross-cultural settings.

CulturaSim solves this problem by simulating realistic conversations using AI and providing feedback to improve communication skills.

---

## 🚨 Problem Statement

In today's global world:

* Over **70% of professionals work in multicultural teams**
* Many students and job seekers feel **unprepared for international communication**
* Lack of cultural awareness can lead to **misunderstandings and lost opportunities**

Traditional language learning tools focus on vocabulary but **do not train real-life cultural communication skills**.

---

## 💡 Our Solution

CulturaSim provides an **AI-powered conversation simulator** where users can:

* Practice real-world communication scenarios
* Interact with AI-generated cultural personas
* Receive feedback on communication style
* Improve confidence in professional conversations

---

## ✨ Features

### 🗣 AI Conversation Simulation

Users can interact with AI characters representing different cultures.

### 🌎 Cross-Cultural Training

Learn how communication styles vary across countries and cultures.

### 🎤 Voice Interaction

Speak naturally and receive AI responses in real time.

### 📊 Communication Feedback

The system analyzes responses and provides improvement suggestions.

### 🎭 Scenario-Based Practice

Practice situations like:

* Job interviews
* Business meetings
* Networking events
* International teamwork

## 🚀 Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/shreyashkashid54-ind/XR_samvaad.git
```

### 2️⃣ Navigate to the project folder

```bash
cd XR_samvaad
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Add API Keys

Create a `.env` file and add:

```env
GEMINI_API_KEY=your_gemini_api_key
DEEPGRAM_API_KEY=your_deepgram_api_key
```

### 5️⃣ Run the project

```bash
npm start
```
